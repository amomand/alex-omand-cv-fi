This repository contains my public CV, published via GitHub Pages. A small CI workflow also generates a matching PDF for convenience.

### Live CV
- The most up-to-date version of my CV lives in `docs/index.md`.
- It is published at alex.omand.uk using GitHub Pages.

### PDF version
- The PDF is generated by a separate GitHub Actions workflow on pushes to `main`.
- The workflow renders the published CV to PDF and commits the result back into `docs/` so it is served by Pages.
- Any PDF-specific formatting lives in print-scoped CSS so the web version stays clean.

### Updating the CV
- Edit `docs/index.md` to update the live CV.
- Changes pushed to `main` automatically rebuild and redeploy the site and PDF.

---

This repo is intentionally simple and lightweight, treating the CV as a versioned, deployable artefact rather than a static document.
